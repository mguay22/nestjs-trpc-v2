# NestJS Quickstart Guide
<Link href="https://nestjs.com/" rel={"nofollow"} target={"_blank"} className={"underline text-primary"}>NestJS</Link> is a progressive Node.js framework for building efficient, reliable, and scalable server-side applications. It uses TypeScript by default, which enhances productivity and maintainability. NestJS incorporates elements from Object-Oriented Programming, Functional Programming, and Functional Reactive Programming.

### Why Choose NestJS?

- **Modular Architecture**: Facilitates code organization and reuse.
- **TypeScript Support**: Out-of-the-box TypeScript support ensures type safety and development efficiency.
- **Dependency Injection**: Built-in dependency injection for better application structure and testing.
- **Ecosystem**: A rich ecosystem of tools and libraries, including support for GraphQL, WebSockets, and more.

### Official Documentation
This guide goes through the basics of NestJS. for a more indepth look, check out their <Link href="https://docs.nestjs.com/" rel={"nofollow"} target={"_blank"} className={"underline text-primary"}>official documentation</Link>.

<Steps>
  <Step>
  ### Installation

  To start using NestJS, you need to install the CLI:

  ```bash title="Terminal"
  npm i -g @nestjs/cli
  ```
  </Step>

  <Step>
  ### Creating a New Project

  Generate a new NestJS project using the CLI:

  ```bash title="Terminal"
  nest new project-name
  ```
  </Step>

  <Step>
  ### Running the Application

  Navigate to the project directory and start the application:

  ```bash title="Terminal"
  cd project-name
  npm run start
  ```
  </Step>
</Steps>

Awesome! You should be all set to integrate **NestJS tRPC** into your application!
